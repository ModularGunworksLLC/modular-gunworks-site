# Modular Gunworks LLC

Welcome to the official GitHub Pages repository for **Modular Gunworks LLC**, a veteran-owned Federal Firearms Licensee (FFL) based in Huntsville, Alabama.

This repository powers the public-facing website at:

**https://modulargunworks.com**

Our mission is to provide a clean, reliable, and professional storefront experience for firearms, suppressors, optics, ammunition, and tactical gear. This site is built with a modular architecture designed for long-term scalability, audit-ready compliance, and seamless integration with distributor data.
